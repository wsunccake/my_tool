# Operation


```
linux:~ $ git clone http://
linux:~ $ pip install PyBuilder
```


```
linux:~/my_tool $ pyb -h
linux:~/my_tool $ pyb -t
```


## Install Required Module

```
linux:~/my_tool $ pyb install_dependencies
```


## Build Package

```
linux:~/my_tool $ pyb
```


## Run Unittest

```
linux:~/my_tool $ pyb run_unit_tests
```
